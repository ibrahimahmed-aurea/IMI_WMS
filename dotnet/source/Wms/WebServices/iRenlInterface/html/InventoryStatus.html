Segment InventoryStatus

Description:

Inventory Status
Setup of available Hold Codes in IMI Warehouse

Attributes:

TagMethodTypeOracle TypeMandatoryNote
OPCODEN, C, RstringVARCHAR2(1) Operation Code
0 - No Operation
1 - New
2 - Change
3 - Remove
Note! An operation code is only useful if at least one tag supports the method
InventoryStatusCodeN, C, RstringVARCHAR2(8)N, C, RHold Code
System defined values (BLOCKTYPE = N):
00 = Not on Hold
P1 = Package Change
E1 = Held by a Lift Truck
E2 = Item Load Missing
You can add more in user defined Type.
InventoryStatusNameN, CstringVARCHAR2(35)NHold Code Description
InventoryStatusAutoResetN, CstringVARCHAR2(1)NRelease Hold Automatically
Means that the hold will be removed from the Item Load/Pick Location when the registered alarm Date for the hold has passed.
