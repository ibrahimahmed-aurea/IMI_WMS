Setup of available Hold Codes in IMI Warehouse

Column Method Type Mand. Note
InventoryStatusCode N, C, R c8 * A Hold Code may be applied on an Item Load or a Pick Location to temporarily exclude it from further processing. System defined values (BLOCKTYPE = N):
00 = Not on Hold
P1 = Package Change
E1 = Held by a Lift Truck
E2 = Item Load Missing
You can add more in user defined Type.
InventoryStatusName N, C c35 N The description/name of the Hold Code.
InventoryStatusAutoReset N, C c1 N The hold will be removed from the Item Load/Pick Location when the registered Alarm Date for the hold has passed.
