Host Application Programming Interface (HAPI)

Message Direction

Each message is either inbound (to IMI Warehouse) or outbound (from IMI Warehouse). The direction of a specific message is shown in the header of each message description as (In) or (Out).

Method

The codes in the Method column can assume the following values:

N
 - new object
C
 - change object
R
 - remove object.

Example: The signature for the Remove operation can be determined by looking at all the attributes that have a 'R' in the method column.

Type

Shows the data type and length as declared in IMI Warehouse.

Mandatory

The codes in the Mand column in the table should be interpreted as:

*
 - always mandatory
-
 - never mandatory
code
 - mandatory for this/these methods. Example N,C means mandatory for New and Change.


For outbound messages Mandatory is not displayed since it implementation dependant.
