Segment Confirm

Description:

Confirmation Message

Attributes:

TagMethodTypeOracle TypeMandatoryNote
OPCODEN, C, RstringVARCHAR2(1) Operation Code
1 - New
TransactionIdNstringVARCHAR2(35) Transaction Identity
The identity of the transaction in the Communication Partners system that sent the original incoming message,
that has been processed and whose result (TRANSACTION_STATUS) is included in this confirmation message.
TransactionStatusNstringVARCHAR2(1) Transaction Status
The result of the processing of the incoming message transaction:
0 = Unsuccessful (Error)
1 = Successful.
ErrCodeNstringVARCHAR2(12) Error Code
Only used when the Transaction Status = Error.
The error Code is either received from Oracle or set to -20100 when
the error is raised by the application.
ErrMsgNstringVARCHAR2(1024) Error Message
The error message. Only used when the Transaction Status = Error.
The error message is either received from Oracle or raised by the application.
