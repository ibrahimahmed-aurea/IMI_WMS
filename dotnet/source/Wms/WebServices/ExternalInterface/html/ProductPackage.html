Segment ProductPackage

Description:

Product Package

Attributes:

TagMethodTypeOracle TypeMandatoryNote
OPCODEN, C, RstringVARCHAR2(1) Operation Code
0 - No Operation
1 - New
2 - Change
3 - Remove
Note! An operation code is only useful if at least one tag supports the method
ProductNumberN, C, RstringVARCHAR2(35)N, C, RProduct Number
The identity of the Product.
PackageIdentityN, C, RstringVARCHAR2(17)N, C, RPackage Identity
The identity of the Package / Unit of Measure.
PackageNameN, CstringVARCHAR2(35) Package Description
The description of the Package / Unit of Measure.
BasePackageIdentityN, CstringVARCHAR2(17)NLowest Unit of Measure
The identity of the Lowest Unit of Measure / Base Package of the Product that can be handled.
NumberofBasePackagesN, CNullableNUMBER(20,6)NLowest Unit of Measure Quantity
The number of Lowest Unit of Measure that makes up the Unit of Measure / Package.
NumberofConsumerPackagesN, CNullableNUMBER(20,6) No of Consumer Packages
The number of consumer packages that are included in the Package / Unit of Measure.
WeightN, CNullableNUMBER(16,6) Weight
The total Weight of the Package / Unit of Measure.
NOTE! This Weight has to be greater than zero (0) if you want to store the Package in an Area with Dynamic Size
and place them on top of each other with consideration to the Top Load.
HeightN, CNullableNUMBER(9,4) Height
The heigth of the Package / Unit of Measure.
LengthN, CNullableNUMBER(9,4) Length
The length of the Package / Unit of Measure.
WidthN, CNullableNUMBER(9,4) Width
The width of the Package / Unit of Measure.
VolumeN, CNullableNUMBER(19,9) Volume
The total Volume for the Package / Unit of Measure.
PMTYPID_INN, CstringVARCHAR2(35) Packing Material Type at Inbound
Packing Material Type that is always connected to the Product Package when receiving. Can be empty <null>.
PMTYPID_OUTN, CstringVARCHAR2(35) Packing Material Type at Outbound
Packing Material Type that is always connected to the Product Package when picking / shipping. Can be empty <null>.
LayerPackageN, CstringVARCHAR2(1) Layer Pallet Package
Controls (1=Yes,0=No) if the Package is a layer package in a pallet.
Used for calculating the number of and height of Item Load, e.g. when receiving goods.
VolumeFillPercentN, CNullableNUMBER(3) Case Pick Volume (%)
The fill factor of the Package / Unit of Measure on a Pick Load Carrier.
This is normally empty <null> which has the same effect as 100%.
For example, balls will take up more volume in the pick load carrier (Case Pick Volume % is greater than 100)
while flower pots that can be piled one onto the other take up less volume (Case Pick Volume % is less than 100).
BarCodeN, CstringVARCHAR2(20) Bar Code
Bar Code used in outgoing ASN.
BarCodeTypeN, CstringVARCHAR2(7) Bar Code Type
The types of Bar Codes that may be used:
DUN = Dispatch Unit Number
EAN = European Article Number
SCC = Shipping Container Code
UPC = Universal Product Code
PalPakN, CstringVARCHAR2(1) Pallet Package
Indicates (1=Yes,0=No) if the Package is a Pallet Package.
Used for calculating the number of and height of Item Load, e.g. when receiving goods.
PalPickN, CstringVARCHAR2(1) Pallet Pick allowed
Controls (1=Yes,0=No) if the Package or unit can be picked as a pallet.
CartypidN, CstringVARCHAR2(3) Load Carrier Type Identity
The identity of the Type of Load Carrier. The Item Load's Load Carrier Type
is the type of Load Carrier the Package normally exists on.
CarCartypidN, CstringVARCHAR2(3) Movement Load Carrier Type
The type of Movement Load Carrier the Load Carrier normally exists on.
NoPaksCarCarN, CNullableNUMBER(3) Number of Load Carriers
States how many of the Item Load's Load Carriers will fit on the Movement Load Carrier.
NetWeightN, CNullableNUMBER(16,6) Net Weight
The Net Weight for a Product with this Package.
