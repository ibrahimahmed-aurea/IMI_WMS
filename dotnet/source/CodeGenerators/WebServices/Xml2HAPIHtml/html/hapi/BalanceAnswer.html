PickLocationQuantity N n20,6 The Pick Location quantity is calculated in two steps (A and B below) depending on if Trace exists for the Pick Location or not. Only Pick Locations of type Fixed and Temporary is handled here. For Pick Location of type Floating or Station the balance has already been included in Free Quantity calculation.
A: Total quantity for Pick Location with NO trace activated =
+ Quantity for Pick Locations with NO Trace.
- Pick Locations with InventoryStatusCode <> Not on Hold.
B: Total quantity for Pick Location with trace activated =
+ Quantity for Pick Locations with Trace.
- Pick Locations with InventoryStatusCode <> Not on Hold.
The PickLocationQuantity = A + B.
PickedQuantity N n20,6 The picked quantity =
+ The quantity in Customer Orders with the status = Pick Completed.
+ The quantity in Case Pick Order Lines with the status = Finished, but not yet reported to the Host system.
+ The quantity in Pallet Pick Order Lines with the status = Finished, but not yet reported to the Host system.
- Transit goods.
TopickQuantity N n20,6 The not yet picked quantity (reserved quantity) =
+ The Ordered quantity from Customer Orders
- The quantity in Customer Orders with the status = Pick Completed.
- The quantity in Case Pick Order Lines with the status = Finished, but not yet reported to the Host system.
- The quantity in Pallet Pick Order Lines with the status = Finished, but not yet reported to the Host system.
- Transit goods.
CustomerReservedQuantity N n20,6 The quantity that is reserved for Customer Order (including transit goods that hasn't been reported) =
+ Item Loads with the status = Reserved.
- Item Loads with InventoryStatusCode <> Not on Hold.
- Not available goods (not received ASN Load Carrier and reported picked goods).
- Goods from NOT finished Receipts.
ReservedForReplenQuantity N n20,6 The quantity that is reserved for Pick Replen-ishment =
+ Item Loads with the status = Reserved and the hold code = Not on Hold.
- Not available goods (not received ASN Load Carrier and reported picked goods).
- Goods from NOT finished Receipts.
- Quantity reserved for Customer Order of type transit goods.
