Segment InventoryStatus

Description:

Inventory Status
Setup of available Hold Codes in IMI Warehouse

Attributes:

TagMethodTypeOracle TypeMandatoryNote
OPCODEN, C, RstringVARCHAR2(1) Operation Code
0 - No Operation
1 - New
2 - Change
3 - Remove
Note! An operation code is only useful if at least one tag supports the method
InventoryStatusCodeN, C, RstringVARCHAR2(8)N, C, RHold Code
A Hold Code may be applied on an Item Load or a Pick Location
to temporarily restrict its processing. System defined values (BLOCKTYPE = N):
00 = Not on Hold
E1 = Held by a Lift Truck
I1 = Inspection
P1 = Package Change
RA = Reserved for Assembly
VR = Return to Vendor
You can add more Hold Codes in the Configure Hold Code dialog.
InventoryStatusNameN, CstringVARCHAR2(35)NHold Code Description
The description / name of the Hold Code.
InventoryStatusAutoResetN, CstringVARCHAR2(1)NRelease Hold Automatically
The hold will be removed from the Item Load / Pick Location
when the registered Alarm Date for the hold has passed.
