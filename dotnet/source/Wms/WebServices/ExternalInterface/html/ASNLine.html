Segment ASNLine

Description:

Advanced Shipping Notice Line
Specifies the product that should be received.

Rules for deciding Owner of the goods:
1. If the ASN Line references an existing Purchase Order, the Owner is fetched from the Purchase Order.
2. If the ASN Line contains a valid Owner, the Owner stated in the ASN is used.
3. (Only for Stock Replenishment.) If the Client configuration has a default Owner configured for the Warehouse, the Owner is fetched from the Client configuration.
Rule for Customer Returns received via ASN:
- If there is a default Owner on the Client configuration, that Owner is always used.

PurchaseOrderNumberNstringVARCHAR2(35) Purchase Order Number
If the ASN references a Purchase Order Line that exists in IMI Warehouse, the ASN is updated with the following information.
- Sell To Customer reference.
- Vendor Identity.
- Owner Identity.
- Customer Order Line reference.
- Original Purchase Order Line reference.
- Original Customer Reference.
- Catch Measure requirements.
- Customer Order Type.
Purchase Order usage in Transit ASN:
- If the ASN references a Purchase Order Line that exists in IMI Warehouse, the Purchase Order Line is assumed to be obsolete when the transit goods has been received in the warehouse.
- I.e. the Purchase Order Line reference is not sent to next node when the transit goods is shipped from the warehouse.

CatchMeasureLowerToleranceNNullableNUMBER(20,6) Lower Limit for Measured Quantity
The minimum accepted Measured Quantity per ordered unit.
It is used to check that the registered Measured Quantity is within the limits of reason.
This tolerance quantity is transferred from the Host System.
Example: The Ordered Quantity is 20 cheeses of 700 gram each.
The Lower Limit for Measured Quantity = 550 gram. When the user measures all 20 cheeses,
the system will accept any registered weight that is above 11,0 kg (=20x550g).

The Upper Limit for Measured Quantity = 770 gram. When the user measures all 20 cheeses,
the system will accept any registered weight that is below 15.4 kg (=20x770g).
